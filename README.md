## 🧑‍💻 How to Use

### 1. Clone the Repo

```bash
git clone [https://github.com/htpsashu404/myPortfolio.git]
cd your-repo-name
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the App

```bash
npm start
```

### 4. Build for Production

```bash
npm run build
```
